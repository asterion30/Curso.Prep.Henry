



Una variable, es como un vaso, puede contener jugo, gaseosa, algún licuado, la diferencia con un vaso real es que lo fabricas vos, lo creas para que contenga lo que quiera.
Un string, es una cadena en ingles, como los eslabones están unidos y forman parte de una misma cosa ("Cosa" es un string ), esta cadena puede cortarse con las herramientas adecuadas, o puede unirse a otras cadenas.
Una función es un fragmento de código que tiene un objetivo específico, es decir una serie de tareas que puedo ejecutar al usarla.
Ejemplo antes creamos un vaso, el vaso no tiene nada, podría crear una función que llene el vaso con agua.

function llenar (vaso){
    console.log("El" + vaso + " esta lleno de agua");
}

La funciones puede recibir valores, argumentos, para a hacer algo con ellos, puede ser que quiera utilizar la función llenar para otra cosa.
Por ejemplo llenar una jarra con jugo, o con agua.
function llenar (recipiente, contenido) {
    console.log ("El " + recipiente + " está lleno de " +  contenido);
} 

Las funciones pueden recibir o no argumentos, puedo tener una función para calmar la sed por ejemplo.
function calmarSed(){
    //Tomo un vaso con gaseosa y lo lleno
    var vaso;
    var gaseosa;

    var tomar= vaso + gaseosa;
    
    }

De esta forma esa funciona aparentemente calma la sed, pero nadie lo sabe, porque ella no informa que esto se ha hecho, es decir no devuelve un resultado de la invocación de su magia.
para ello podemos devolver el resultado retornando tomar:
function calmarSed(){
    //Tomo un vaso con gaseosa y lo lleno
    var vaso;
    var gaseosa;
    
    var tomar= vaso + gaseosa;
    return tomar;
    }
Tambien  podemos decir con un condicional si algo se ejecute o no con if:
function llenar (recipiente, contenido) {
   //Lleno el recipiente concatenando el contenido.
   var llenado= recipiente + contenido;
   //Si la variable llenado no es numo devuelvo verdadero
   if llenado != null{
       console.log ("Recipiente completo");
       return true;
   }
   else {
       console.log ("Recipiente no completo");
   }
}
