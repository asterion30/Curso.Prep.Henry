Una funcion es un fragmento de codigo que tiene un objetivo especifico, es decir una serie de tareas que puedo ejecutar al usarla.
Ejemplo antes cree un vaso, el vaso no tiene nada, podria crear una funcion que llene el vaso con agua.

}

De esta forma esa funciona aparentmente calma la sed, pero nadie lo sabe, porque ella no informa que esto se ha echo, es decir no devuelve un resultado de la invocaci[on de su magia.
para ello podemos devolver el resultado retornando tomar:
function calmarSed(){
    //Tomo un vaso con gaseosa y lo lleno

    var gaseosa;
    
    var tomar= vaso + gaseosa;
    return tomar;
    }
